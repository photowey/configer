# `configer`

A simple configuration management implemented in Rust.

## `Changelog`

- `v0.1.0`

    - Support `set/get`.

- `v0.2.0`
    - Support `try_xxx` functions.

- `v0.3.0`

    - Support read `toml` file.

- `v0.4.0`
    - Support build `ConfigerEnvironment` by `ConfigerEnvironmentBuilder`
